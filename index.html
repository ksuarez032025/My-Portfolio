AI Recipe Generator

The AI Recipe Generator is a web-based application that allows users to generate custom recipes based on prompts like available ingredients or dietary preferences. Built using HTML, CSS, and JavaScript, the project emphasizes API interaction with an AI model, along with dynamic DOM updates to display creative recipe outputs. The project highlights practical use of AI in a consumer-facing tool and introduces prompt optimization to improve response relevance.

Weather App

The global weather app was designed to retrieve and display live weather and forecast data using SheCodes APIs. The project demonstrated how to manage asynchronous JavaScript operations to fetch both current and future weather data. Users could view conditions by location with a clean interface and responsive layout.
